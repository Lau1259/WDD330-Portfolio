Code Highlights - localStorage

localStorage data is specific to the protocol of the document. In particular, for a site loaded over HTTP (e.g., http://example.com), localStorage returns a different object than localStorage for the corresponding site loaded over HTTPS (e.g., https://example.com).

Available functions:

  • setItem This adds data items to the local storage object.
  • getItem This retrieves items from the local storage object.
  • removeItem This removes items from the local storage object.
  • clear This removes all items from the local storage object.

Example:

Example of retrieving items from local storage. 
function loadStory(){
  var storyName = document.getElementById(“name_input”).value
  var storyHTML = localStorage.getItem(storyName)
  document.getElementById(“story_editor”).value = storyHTML;
}
Example of saving items to local storage. 
function saveStory(){
  var storyName = document.getElementById(“name_input”).value
  var storyHTML = document.getElementById(“story_editor”).value
  localStorage.setItem(storyName, storyHTML)
}

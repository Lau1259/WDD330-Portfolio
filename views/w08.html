CSS Transforms & Transitions

CSS transforms and transitions are one way of achieving basic animations with CSS. Transforms can be used to move items, change their color, or even change their scale.

I'll exemplify this with the card below

The card above uses CSS transform and transition properties to trigger the simple animation above. The transform is used to scale the label that says Chihuahua and the transition property is used to make the label animation last .35s

Additionally, the above animation uses the transition-delay property on the background color for the text. I wanted the animation to trigger after the label scales so I used a delay to make sure it is triggered at the right moment.

Canvas is a way to draw using JavaScript. It can be used for games that need to update the visuals often. Canvas has some benefits like being able to save a canvas to a png or jpg. SVG

SVG is a way to display digital graphics on the web. It allows you to create basic shapes such as a rectangle and a circle. It also allows the user to draw a path; This is most often done using special software such as Adobe Illustrator or Inkscape.

Week 3 Notes

This

In JavaScript the this keyword refers to the current object. This is important to take into account because it adds a level of flexibility to functions. Using this makes it easier to reuse a function across various objects.

Example: function getName(object){
let name = this.name;
return name;
}

Objects

JavaScript objects are basically the equivalent of a python dictionary. It allows you to save key value pairs.You can also assign specific functions to an object. When an object is assigned a function we refer to it as a method.

A JavaScript object looks like this: let object = {name: "someString"};

Alternatively, you can create a blank object using the following syntax: // Using shorthand
let obj1 = {};

// Using the constructor function
let obj2 = new Object();

Document Object Model (DOM)

The DOM allows us to hook to diffent parts of our page. This is very useful when you want to manipulate something. Common ways to select items in the DOM include: // By element id
let item1 = document.getElementById('myId');

// By class name let items1 = document.getElementByClassName('myClass');

// Using query select. This essentially allows you to reference objects using css selectors
let item1 = document.querySelect(#myId);

//You can also use querySelectAll to get multiple values let images = document.querySelectAll(img);

Events

JavaScript events are a way to keep track of changes that may happen in the browser. This includes clicks, moseovers, screen taps, loading, etc. These events must be tracked by adding an event listener. These event listeners track changes and can call a callback function once an event is detected. * NOTE: Inline click events should be avoided.

Basic event syntax: // Adding an event listener to the document body
document.body.addEventListener('click',doSomething)
